Statistical Programming Assignment in SAS
Explanations of Project Code
External Datasets used in program:
birds.dat
donner.dat
ratmaze.dat

Assignment Objective: In this assignment I used a two-way analysis of variance test to see if changes in levels of categorical predictor variables would cause change in the mean of a quantitative response variable. I then used an interaction plot to try an explain if there was any interaction between the predictor and response variable. In the assignment I also used a t-test to analyze the mean output variable of two independent groups to see if there were any differences between one another. Another test statistic that was used in this assignment was the kappa coefficient. The kappa coefficient was used to measure the level of agreement between three cutoff values as well as between two independent groups. Next in the assignment I used a logistic regression procedure to predict a binary outcome of a categorical group based on two other independent/predictor variables. Lastly, in the assignment I used a frequency test to see if any change occurred on a numeric variable based on different levels of an ordinal predictor variable of the dataset. After which I tested for association between the two variables with a chi-square test.

Assignment Objective: In this assignment I compared data from four datasets also known as Anscombe’s quartet presented by French statistician Francis Anscombe. I used these datasets to calculate the mean, variance, correlation, and linear regression to show how nearly identical simple descriptive statistics, can have very different distributions.

One-way ANOVA Analysis in SAS
Two-way ANOVA Analysis in SAS
Explanations of Assignment Code

Assignment Objective: This assignment contains two SAS programs that conduct statistical analyses. The first uses one-way ANOVA test to test three different types of drugs and their affect on patient’s good cholesterol (HDL) during a study. The second program continues the analysis of a patient’s HDL levels based on drug type, but also adds another interaction term to the linear model of a person’s gender. Since the design of the study was unbalanced, I now had to use a PROC GLM procedure in SAS to conduct a two-way ANOVA analysis to see what affects each drug type had on a person’s HDL level with the addition of the interaction term of a person’s gender. I also checked to see if there was in fact any interaction between the gender variable and the drug type variable of the model, as well.
